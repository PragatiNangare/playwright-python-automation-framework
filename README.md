# Playwright Python Automation Framework

## 📌 Overview

This project is a UI automation framework built using **Playwright with Python**, following the **Page Object Model (POM)** design pattern.

The framework is being developed incrementally by covering different web automation scenarios from:
https://the-internet.herokuapp.com/

---

## 🚀 Features (Completed So Far)

* Page Object Model (POM) structure
* Form handling (inputs & login)
* Checkbox interactions
* Dropdown handling
* JavaScript Alerts (Alert, Confirm, Prompt)
* Multi-tab (window) handling

---

## 🛠️ Tech Stack

* Python
* Playwright
* Pytest
